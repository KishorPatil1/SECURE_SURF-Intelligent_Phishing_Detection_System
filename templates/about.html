About Us

This project focuses on developing an intelligent phishing attack detection system using deep learning techniques. With the rapid increase in online threats, phishing has emerged as one of the most common and dangerous cybercrimes. To address this, our system leverages two powerful deep learning models—Convolutional Neural Networks (CNN) and Multi-Layer Perceptrons (MLP)—to accurately classify URLs and web content as either legitimate or phishing. The CNN model is used to automatically extract spatial features from the input data, while the MLP model captures deeper, non-linear relationships for improved prediction accuracy. By training and evaluating both models on a labeled phishing dataset, we aim to create a robust detection mechanism that enhances online security and minimizes user exposure to fraudulent websites.
